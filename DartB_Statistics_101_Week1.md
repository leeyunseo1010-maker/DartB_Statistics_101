# 1️⃣ 개념 정리 
## 01. 데이터를 분석하다

```
✅ 학습 목표 :
* 데이터에 대해 이해한다.
* 데이터 분석에 대해 이해한다.
```
데이터의 이해: 데이터는 우리 주변의 현상을 숫자나 문자로 기록한 기초적인 재료이다

데이터 분석의 의미: 단순히 쌓여있는 데이터를 정리하는 것에 그치지 않고, 그 안에서 유의미한 해석을 이끌어내는 과정


## 02. 통계학의 역할

```
✅ 학습 목표 :
* 산포에 대해 설명할 수 있다.
```

데이터가 평균에만 몰려 있는지, 아니면 넓게 퍼져 있는지 파악하는 것이 분석의 핵심


## 03. 통계학의 전체 모습
```
✅ 학습 목표 :
* 기술통계와 추론통계의 차이점을 설명할 수 있다.
* 통계적 추론과 가설 검정에 대해 이해한다.
```
기술통계(Descriptive Statistics): 현재 내가 가지고 있는 데이터를 그래프나 수치로 정리하여 '묘사'하는 기초 단계

추론통계(Inferential Statistics): 전체(모집단)를 다 조사할 수 없을 때, 일부(표본)를 통해 보이지 않는 전체를 '예측'하는 고도의 단계

가설 검정: "이 차이는 우연일까, 아니면 의미 있는 결과일까?"라는 질문에 통계적 확률로 답을 내리는 과정


## 04. 데이터 분석의 목적과 알고자 하는 대상

```
✅ 학습 목표 :
* 데이터 분석의 목적에 맞게 ‘알고자 하는 대상(모집단)’을 명확히 정의하는 것이 왜 중요한지 이해한다.
```

문제 정의: 분석을 시작하기 전 "우리가 진짜 알고 싶은 집단은 누구인가?"를 정하는 것이 가장 중요함

타겟의 명확화: 모집단을 잘못 정의하면 아무리 정교한 분석 기법을 써도 잘못된 인사이트를 얻게 되어 시간, 비용, 자원을 낭비하게 될 수도 있음


## 05. 모집단

```
✅ 학습 목표 :
* 모집단의 개념을 명확히 설명할 수 있다.
```

모집단(Population): 조사 목적에 포함되는 분석 대상 전체 집단을 의미

- 유한모집단 vs 무한모집단: 개수를 셀 수 있는 집단(우리 학교 학생 수)뿐만 아니라, 이론적으로 끝이 없는 집단도 모집단이 될 수 있음

- 유한모집단: 모집단 중 한정된 요소만 포함하는 것
- 무한모집단: 모집단 중 포함된 요소의 개수가 무한한 것
- 유한모집단의 예시: 특정 년도의 한국인 명수
- 무한모집단의 예시: 신약 개발과정에서 미래에 이 약을 복용하게 될 환자들도 포함


## 06. 모집단의 성질을 알다 

```
✅ 학습 목표 :
* 전수조사와 표본조사의 개념적 차이와 각각의 필요성을 설명할 수 있다.
* 표본에 대해 이해한다.
```

1. 전수조사

- 모집단 전체를 다 조사하는 방법

- 가장 직접적이지만 시간, 비용, 현실적 제약이 큼

2. 표본조사

- 모집단 일부만 조사하는 방법

- 현실적으로 더 많이 사용됨

- 대신 표본이 모집단을 잘 대표해야 함

3. 표본

- 모집단에서 뽑은 일부 데이터

- 우리가 실제로 데이터분석 과정에서 분석하는 대상

- 이 표본으로부터 모집단의 성질을 추정하려면, 표본이 치우치지 않아야 함

# 2️⃣ 확인 문제

## 문제 1.

> **🧚Q. 한 회사가 직원들의 ‘업무 만족도’를 파악하기 위해 설문조사를 실시했다.
전체 직원 1,000명 중 50명에게만 설문을 진행했고, 그 결과 평균 만족도는 4.5점(5점 만점)이었다.
회사는 이를 근거로 “우리 회사 직원들은 대체로 업무 만족도가 높다”고 결론 내리려 한다.**
>
> **이 결론이 타당한지 판단하기 위해 가장 먼저 확인해야 할 것은 무엇인가?**

~~~
1️⃣ 조사 대상 50명이 전체 직원을 대표하는 표본인가?
2️⃣ 만족도의 평균값만으로 결론을 내려도 되는가?
3️⃣ 설문 문항이 객관적으로 구성되었는가?
4️⃣ 이 결과를 모집단(전체 직원)으로 일반화해도 되는가?
~~~



<!--학습한 개념을 활용하여 자유롭게 설명해 보세요.-->

```
정답: 1️⃣ 조사 대상 50명이 전체 직원을 대표하는 표본인가?
근거: 50명의 표본으로 1,000명 전체를 추론한 '추론통계' 이므로, 표본의 대표성을 먼저 확인해야 합니다.
표본이 특정 부서나 직급에 치우치면 4.5점이라는 결과가 전체 직원의 만족도를 제대로 반영하지 못할 수 있습니다.
즉, 표본이 왜곡되면 통계적 결론도 함께 왜곡됩니다.
전수조사가 어려울 때 표본조사는 유용하지만, 무작위성과 대표성이 보장되어야 합니다.
그래서 점수 자체보다 먼저 표본 추출 과정이 적절했는지를 살펴봐야 합니다.

```
